# Meme-generator
# 🎭 Meme Generator

A responsive and interactive **Meme Generator Web Application** developed using **HTML, CSS, AngularJS, and JavaScript**.
This project allows users to create custom memes by adding personalized top and bottom text to images in real-time.

The application is simple, lightweight, and user-friendly, making meme creation fast and entertaining.



# 📌 Project Overview

The Meme Generator is designed to help users create funny and creative memes instantly. Users can either use an online image URL or a meme template and customize it with text styling options.

Memes are an important part of internet culture and online communication. This project demonstrates how web technologies can be used to build interactive and visually engaging applications.



# 🚀 Features

* 🖼️ Add image using URL
* ✍️ Add custom top and bottom text
* 🎨 Dynamic text styling
* 📏 Adjustable text size using sliders
* ⚡ Real-time meme preview
* 📱 Responsive user interface
* 🌐 Simple and lightweight web application
* 🎭 Canvas-based meme rendering

# 🧠 Working of the Project

1. User enters an image URL.
2. The image is loaded onto the HTML canvas.
3. User enters top and bottom text.
4. AngularJS watches the text changes dynamically.
5. Text is rendered over the image in real-time.
6. Users can adjust text sizes using sliders.
7. Final meme is displayed instantly.
